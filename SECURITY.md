# Security Policy

## Supported Versions

This is a research documentation repository. Security issues are primarily:
- accidental disclosure of private notes
- malicious prompt payload examples
- unsafe operational instructions

Latest canonical docs are considered supported.

## Reporting a Vulnerability

Please open an issue titled:
`[security] short summary`

Include:
- affected file(s)
- risk description
- reproduction steps
- suggested mitigation

If sensitive, share minimal details publicly and request private follow-up.

## Disclosure Preference

Please avoid public zero-day style disclosure of sensitive vectors.
Give maintainers a reasonable window to patch or remove risky content first.
